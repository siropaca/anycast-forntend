# おすすめのチャンネル一覧 (`/channels`)

ホームの「おすすめのチャンネル」セクションの「もっと見る」遷移先。認証不要でアクセスできる。

## 表示内容

- **ページタイトル**
  - 「おすすめのチャンネル」
- **チャンネル一覧**
  - おすすめチャンネルをグリッド表示
  - チャンネルアートワーク・チャンネル名・カテゴリ名
  - 各チャンネルはチャンネル詳細ページへのリンク

## アクション

| アクション | 未ログイン | ログイン済み |
|------------|:----------:|:------------:|
| チャンネル詳細ページへ遷移 | ✅ | ✅ |

## API エンドポイント

- `GET /recommendations/channels` — おすすめチャンネル一覧取得

## SEO

- インデックス対象（`robots: { index: true }`）
- OGP: 固定タイトル・説明文を設定
