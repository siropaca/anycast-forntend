# 再生リスト詳細 (`/library/playlist/:id`)

再生リスト内のエピソード一覧を表示するページ。認証必須。

## 表示内容

- **ページタイトル**
  - 再生リスト名
- **エピソード一覧**
  - 再生リストに追加したエピソードを追加順で表示
  - エピソードアートワーク・タイトル・チャンネル名
  - 各エピソードはエピソード再生ページへのリンク
- **エンプティ**
  - 再生リストが空の場合、案内メッセージを表示

## アクション

| アクション | 説明 |
|------------|------|
| エピソード再生ページへ遷移 | エピソードをタップして再生ページへ |
| 再生リストから削除 | エピソードを再生リストから削除 |

## API エンドポイント

- `GET /playlists/{playlistId}` — 再生リスト詳細取得
- `DELETE /episodes/{episodeId}/playlist` — 再生リストから削除
