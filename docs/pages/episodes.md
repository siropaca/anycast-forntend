# おすすめのエピソード一覧 (`/episodes`)

ホームの「おすすめのエピソード」セクションの「もっと見る」遷移先。認証不要でアクセスできる。

## 表示内容

- **ページタイトル**
  - 「おすすめのエピソード」
- **エピソード一覧**
  - おすすめエピソードをグリッド表示
  - エピソードアートワーク・タイトル・チャンネル名
  - 各エピソードはエピソード再生ページへのリンク

## アクション

| アクション | 未ログイン | ログイン済み |
|------------|:----------:|:------------:|
| エピソード再生ページへ遷移 | ✅ | ✅ |

## API エンドポイント

- `GET /recommendations/episodes` — おすすめエピソード一覧取得

## SEO

- インデックス対象（`robots: { index: true }`）
- OGP: 固定タイトル・説明文を設定
